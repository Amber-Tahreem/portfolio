# Amber Tahreem - Developer Portfolio

A cinematic, high-performance personal portfolio designed to showcase full-stack development projects and UI/UX engineering. 

**🔗 [View Live Site](https://your-vercel-link-here.com)**

## 🛠 Tech Stack
This project was built with a focus on modern web technologies and smooth, physics-based animations:
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Typography:** Space Grotesk 
* **Deployment:** Vercel

## ✨ Key Features
* **Cinematic Load Sequence:** Highly choreographed entrance animations using Framer Motion variants and staggered delays.
* **Interactive UI:** Complex SVG clip-path animations and spring-physics interactions.
* **Responsive Design:** Fluid typography (viewport width sizing) and CSS grid layouts that adapt perfectly from mobile to ultrawide monitors.
* **Textured Aesthetic:** Custom SVG noise filters and CSS linear-gradient grids for a premium, tactile interface.
